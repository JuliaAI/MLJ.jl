Third Party Packages

A list of third party packages with integration with MLJ.

Last updated December 2020.

Pull requests to update this list are very welcome. Otherwise, you may post an issue requesting this here.

Packages providing models in the MLJ model registry

  • ScikitLearn.jl
  • DecisionTree.jl
  • MultivariateStats.jl
  • MLJModels.jl
  • MLJLinearModels.jl
  • LIBSVM.jl
  • EvoTrees.jl
  • NaiveBayes.jl
  • MLJFlux.jl (depending on Flux.jl)
  • Clustering.jl
  • ParallelKMeans.jl
  • NearestNeighborModels.jl (depending on NearestNeighbors.jl)
  • PartialLeastSquaresRegressor.jl
  • LightGBM.jl
  • GLM.jl
  • XGBoost.jl
  • BetaML.jl

Providing unregistered models:

  • SossMLJ.jl
  • TimeSeriesClassification

Packages providing other kinds of functionality:

  • TreeParzen.jl (hyper-parameter optimisation strategy)
  • Shapley.jl (feature ranking / interpretation)
  • ShapML.jl (feature ranking / interpretation)
  • Fairness.jl (FAIRness metrics)
