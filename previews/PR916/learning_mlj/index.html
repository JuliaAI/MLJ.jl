New to machine learning?

Try the "Introduction to Statistical Learning" notebooks at Data Science Tutorials, starting with this tutorial. Or have a look at the Julia Data Science book.

Know a little machine learning and just want some MLJ basics?

Read Getting Started.

A practicing data scientist transitioning from another platform?

If, for example, you're coming from scikit-learn or R, we recommend your start with MLJ for Data Scientists in Two Hours.

Try this Lightning Tour.

Learn about Learning Networks.
